Concept: This is the first webpage I made. It is a simple game of hangman based on Game of Thrones. Guess the name of the character based on the banner to collect it.


What did I learn: The point of this game was to practice arrays and objects.


Obstacles: I wanted to keep the names random, and list the banners in the order they were won. This was an tough to learn at first because I did this before learning jQuery and the this.
